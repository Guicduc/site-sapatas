# Plano de SEO

## Norte editorial

O site deve atacar buscas com intenção clara de compra ou especificação. A prioridade não é volume genérico, e sim presença forte em consultas de componente.

## Clusters prioritários

### 1. Produto

- sapata para moveis
- sapata para pe de mesa
- sapata para cadeira
- sapata para tubo redondo
- sapata para tubo quadrado
- sapata plastica sob medida

### 2. Solução

- sapata impressa em 3d

- ponteira para mobiliario

### 3. Objeção e compatibilidade

- como escolher sapata para movel
- sapata para piso delicado

- ponteira com medida especial

## Diretrizes on-page

- um `title` específico por página
- `meta description` orientada a benefício e compatibilidade
- `h1` alinhado à keyword principal
- FAQs ligadas a prazo, cor, medida, material e personalização
- links internos entre famílias, catálogo e projeto especial
